Detailed Description

Image filter/transform uses convolution matrix/kernel.

Construct a new Abstract Kernel Based Image Filter Transform:: Abstract Kernel Based Image Filter Transform object.

Also constructs an "identity kernel", i.e. the center of the matrix is 1, and the rest is 0. For example of a size 2 kernel. 0 0 0 0 1 0 0 0 0

Parameters
sizeSets the size of the class to this size argument. Size refers to the radius of the matrix. E.g. size 3 = 3*2-1 = 5. A 5x5 matrix.
parentPassed to AbstractImageFilterTransform() constructor.
