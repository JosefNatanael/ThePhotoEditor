Construct an edge detection kernel with radius size.

An identity kernel with a dimension ( size * 2 - 1) * ( size * 2 - 1) is constructed. The center kernel pixel weight will be the total number of cells minus 1 with all other weights -1.

Example of an edge detection kernel with size 2. 

-1 -1 -1
-1  8 -1
-1 -1 -1

sizeSize/radius of the kernel.
