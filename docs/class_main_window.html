◆ rerenderWorkspaceArea

void MainWindow::rerenderWorkspaceArea ( const QImage &  image,
int  imageWidth,
int  imageHeight 
)
privateslot

Rerenders the workspace area.

Mechanism to view the cropped/filtered/transformed image:

  1. store current workspaceArea to a temporaryArea
  2. create a new workspaceArea to hold the new cropped image, and set the graphicsView to contain this new workspaceArea
  3. the temporaryArea will then be destroyed when a) destroyed by the destructor, or b) there is a new signal to crop the image. We cannot destroy/reconstruct the workspaceArea in this slot, as this slot is connected (signal-slot connection) to the caller signal in the workspaceArea.
    Parameters
    imageImage to be rendered.
    imageWidthWidth of the image.
    imageHeightHeight of the image.
